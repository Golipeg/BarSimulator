Plugins: Папка Plugins предназначена для хранения внешних плагинов, 
библиотек и другого программного обеспечения, 
которые взаимодействуют непосредственно с ядром Unity. 
Если SDK рекламы представляет собой плагин, 
который напрямую интегрируется с Unity и 
использует его функциональность (например, отображение рекламы внутри игры), 
то Plugins может быть подходящим местом для размещения этого SDK.
