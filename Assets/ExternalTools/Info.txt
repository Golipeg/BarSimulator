ExternalTools: Папка ExternalTools может быть использована для хранения инструментов 
и утилит, которые не взаимодействуют напрямую с ядром Unity, 
но предоставляют вспомогательную функциональность или 
API для вашего проекта. Если SDK рекламы представляет собой внешний инструмент, 
который вы вызываете или используете в вашем проекте
 для управления рекламой (например, для отправки запросов на сервер рекламной сети), 
то ExternalTools может быть более подходящим вариантом.
